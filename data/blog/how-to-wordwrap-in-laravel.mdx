The latest addition to the [Laravel](https://laravel.com/) String API, the wordWrap method, brings enhanced string management capabilities to developers. This method, acting as a wrapper for [PHP's](https://www.php.net/) native wordwrap function, empowers Laravel developers with increased flexibility when handling strings.

## Using the String Facade:

Discover the magic of Laravel's String Facade and its groundbreaking wordWrap method, simplifying the art of string manipulation. Seamlessly wrap your strings to a specified character limit, unveiling neatly formatted and easily readable outputs. This empowers developers with newfound control over string presentation, ensuring a sleek and polished appearance.

```php
<?php

use Illuminate\Support\Str;

$text = "A very long woooooooooooord.";
$wrappedText = Str::wordWrap(string: $text, characters: 8);

echo $wrappedText;
// Output: A very long wooooooo ooooord.

?>
```

## Using Stringable:

Delve into the world of hassle-free string handling through Laravel's Stringable class and the latest addition, the intuitive wordWrap method. Effortlessly apply character limits, transforming the way you manipulate strings for optimal clarity and readability. Stringable emerges as a sleek, concise, and elegant solution, revolutionizing efficient string wrapping within your Laravel applications.

```php
<?php
use Illuminate\Support\Stringable;

$text = "A very long woooooooooooord.";
$wrappedText = str($text)->wordWrap(characters: 8);

echo $wrappedText;
// Output: A very long wooooooo ooooord.

?>

```
