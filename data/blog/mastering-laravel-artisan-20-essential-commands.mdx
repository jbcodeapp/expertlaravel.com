# Overview

<TOCInline toc={props.toc} exclude="Overview" toHeading={2} defaultimage="" />

[Laravel](https://laravel.com/) Artisan is your trusty sidekick in the world of Laravel development. This [command-line tool](https://laravel.com/docs/10.x/artisan) simplifies tasks, boosts productivity, and adds a touch of magic to your projects. In this comprehensive guide, we'll explore 20 essential Laravel Artisan commands that will elevate your development game.

## 1. Creating a New Laravel Project

Creating a new Laravel project is a [breeze](https://laravel.com/docs/10.x/starter-kits#laravel-breeze) with [Artisan](https://laravel.com/docs/10.x/artisan#introduction). Simply use the new command followed by your project name:

```bash
composer create-project --prefer-dist laravel/laravel project-name
```

## 2. Running a Local Development Server

Spin up a local server for your [Laravel app and start](https://laravel.com/docs/4.2/quick#installation) development with:

```bash
php artisan serve
```

## 3. Generating Controllers

Create a new [controller](https://laravel.com/docs/10.x/controllers) to handle your application's logic:

```bash
php artisan make:controller MyController
```

## 4. Generating Models

Generate an [Eloquent](https://laravel.com/docs/10.x/eloquent) model for your database interactions:

```bash
php artisan make:model MyModel
```

## 5. Creating Migrations

[Migrations](https://laravel.com/docs/7.x/migrations) help manage your database schema. Create a new migration with:

```bash
php artisan make:migration create_table_name
```

## 6. Running Migrations

Execute pending migrations to update your database schema:

```bash
php artisan migrate
```

## 7. Rolling Back Migrations

If something goes wrong, you can [roll back migrations](https://laravel.com/docs/10.x/migrations#rolling-back-migrations):

```bash
php artisan migrate:rollback
```

## 8. Creating Seeder Classes

Seed your database with [dummy data](https://laravel.com/docs/7.x/database-testing#using-seeds) using seeders:

```sbash
php artisan make:seeder MySeeder
```

## 9. Seeding the Database

Populate your database with sample data:

```bash
php artisan db:seed
```

## 10. Generating Factories

Factories help create fake data for testing:

```bash
php artisan make:factory MyFactory
```

## 11. Generating Resourceful Controllers

Create [controllers](https://laravel.com/docs/10.x/controllers) with CRUD methods for a resource:

```bash
php artisan make:controller MyResourceController --resource
```

## 12. Creating Custom Artisan Commands

Build custom Artisan commands tailored to your project's needs:

```bash
php artisan make:command MyCustomCommand
```

## 13. Task Scheduling

[Schedule](https://laravel.com/docs/10.x/scheduling) tasks to run at specific times or intervals:

```bash
php artisan schedule:list
```

## 14. Creating Middleware

Generate [middleware](https://laravel.com/docs/10.x/middleware) for custom request handling:

```bash
php artisan make:middleware MyMiddleware
```

## 15. Interactive Tinker

Explore and interact with your application's code using [Tinker](https://laravel.com/docs/10.x/artisan#tinker):

```bash
php artisan tinker
```

## 16. Listing Registered Routes

View a list of all registered routes:

```bash
php artisan route:list
```

## 17. Environment Management

Switch between different [environment](https://laravel.com/docs/10.x/configuration) configurations:

```bash
php artisan env
```

## 18. Generating Events and Listeners

Create events and their associated listeners:

```bash
php artisan make:event MyEvent
php artisan make:listener MyListener --event=MyEvent
```

## 19. Queue Workers

Start a worker to process jobs from the [queue](https://laravel.com/docs/10.x/queues):

```bash
php artisan queue:work
```

## 20. Database Rollbacks

Rollback migrations to undo database changes:

```bash
php artisan migrate:reset
```

## Conclusion :

These 20 Laravel Artisan commands are your toolkit for mastering Laravel development. Whether you're a beginner or an experienced developer, Artisan simplifies tasks, boosts productivity, and makes development a joy. So, open your terminal, harness the power of Artisan, and watch your Laravel projects flourish!

In this guide, we've embarked on a journey to explore the powerful world of Laravel Artisan, a command-line tool that can truly transform your Laravel development experience. From project creation to database management, from code generation to custom command creation, Artisan offers an array of commands to simplify and streamline your workflow.

By mastering these 20 essential Artisan commands, you've gained a strong foundation for efficient Laravel development. Whether you're creating a new project, managing your database, or automating routine tasks, Artisan is your trusted companion.
